## [Model Link](https://api.modellink.online) 官方 AI 媒体生成 API 插件

### 插件概述

[Model Link](https://api.modellink.online) 官方 AI 媒体生成 API 插件是一个集成了多种 AI 媒体生成能力的 Dify 插件，提供图像生成、视频生成、对话生成等多种功能，帮助用户快速创建高质量的 AI 媒体内容。

| Klingus 多图像生成视频查询 | 根据 ID 查询 Klingus 多图像生成视频结果 |

### 工具详细功能说明

#### 1. Nano Banana 图像生成

**功能介绍**：使用 Nano Banana API 生成高质量图像，支持多种模型、宽高比和分辨率。

**参数说明**：
- API 密钥：用于认证的 API 密钥
- 模型名称：选择使用的模型（gemini-2.5-flash-image、gemini-3-pro-image-preview）
- 提示词：用于图像生成的描述文本
- 参考图列表：参考图 URL 列表，使用英文逗号分隔（最多支持 14 张）
- 图像宽高比：选择生成图像的宽高比（1:1、2:3、3:2、3:4、4:3、4:5、5:4、9:16、16:9、21:9）
- 图像分辨率：图像分辨率（仅 gemini-3-pro-image-preview 支持，可选 1K、2K、4K）

**使用场景**：
- 生成创意图片
- 根据参考图生成相似风格的图像
- 生成特定宽高比和分辨率的图像

#### 2. OpenAI 对话

**功能介绍**：OpenAI Chat Completions API 封装，支持标准 OpenAI API 格式，可用于生成文本、进行对话等。

**参数说明**：
- API 密钥：用于认证的 API 密钥
- 对话消息：对话消息列表，每条消息包含角色和内容
- 提示词：用户提示词（messages 的替代方案）
- 模型：模型名称（如 gpt-4o）
- 温度：采样温度（0 到 2），控制输出的随机性
- 最大完成 tokens：生成的完成内容的最大 tokens 数
- Top P：核采样（0 到 1）
- 频率惩罚：降低重复相同文本行的可能性（-2 到 2）
- 存在惩罚：增加引入新主题的可能性（-2 到 2）
- N：完成数量（1 到 128）
- 停止序列：模型停止生成更多 tokens 的字符串数组
- 响应格式：指定响应的格式，可选 text 或 json_object
- 推理努力：指定推理努力程度，影响响应质量和速度（low、medium、high）
- 种子：用于重现性的种子
- Logit 偏差：修改指定 tokens 在完成中出现的可能性
- Logprobs：是否返回 logprobs
- Top Logprobs：要返回的 Top logprobs（0 到 20）

**使用场景**：
- 生成文本内容
- 进行智能对话
- 生成结构化数据（如 JSON）
- 进行多轮对话交互

#### 3. Sora 视频生成

**功能介绍**：使用 Sora API 生成高质量视频，支持多种时长、尺寸和水印设置。

**参数说明**：
- API 密钥：用于认证的 API 密钥
- 模型：模型名称（sora-2、sora-2-pro）
- 提示词：用于视频生成的描述文本
- 时长（秒）：视频时长（sora-2 支持 10、15；sora-2-pro 支持 10、15、25）
- 首帧图：首帧图 URL
- 尺寸：视频尺寸（sora-2 支持 720x1280、1280x720；sora-2-pro 支持所有尺寸）
- 水印：是否添加水印
- 私有：视频是否私有（true 表示视频不会发布，同时无法进行二次编辑）
- 角色视频链接：创建角色需要的视频链接（视频中一定不能出现真人）
- 角色出现时间：视频角色出现的秒数范围，格式 '{start},{end}'（时长 1～3 秒）

**使用场景**：
- 生成创意视频
- 根据首帧图生成视频
- 生成特定时长和尺寸的视频
- 创建角色视频

#### 4. Sora 视频查询

**功能介绍**：根据视频 ID 查询 Sora 视频生成结果。

**参数说明**：
- API 密钥：用于认证的 API 密钥
- 视频 ID：要查询的视频生成任务 ID

**使用场景**：
- 查询视频生成状态
- 获取已生成视频的详细信息

#### 5. Klingus 文本生成视频

**功能介绍**：使用 Klingus API 根据文本生成视频。

**使用场景**：
- 根据文字描述生成视频
- 快速创建文本驱动的视频内容

#### 6. Klingus 图像生成视频

**功能介绍**：使用 Klingus API 根据单张图像生成视频。

**使用场景**：
- 根据静态图像生成动态视频
- 扩展图像内容为视频

#### 7. Klingus 多图像生成视频

**功能介绍**：使用 Klingus API 根据多张图像生成视频。

**使用场景**：
- 根据多张图像序列生成视频
- 创建基于图像集合的视频内容

#### 8. Klingus 文本生成视频查询

**功能介绍**：根据 ID 查询 Klingus 文本生成视频结果。

**使用场景**：
- 查询文本生成视频的状态和结果

#### 9. Klingus 图像生成视频查询

**功能介绍**：根据 ID 查询 Klingus 图像生成视频结果。

**使用场景**：
- 查询图像生成视频的状态和结果

#### 10. Klingus 多图像生成视频查询

**功能介绍**：根据 ID 查询 Klingus 多图像生成视频结果。

**使用场景**：
- 查询多图像生成视频的状态和结果

### 使用示例

#### Nano Banana 图像生成示例

```
API 密钥：sk-xxxxxxxxxxxxxxxx
模型名称：gemini-3-pro-image-preview
提示词：一只可爱的柯基犬在草地上玩耍
参考图列表：https://example.com/corgi.jpg
图像宽高比：16:9
图像分辨率：2K
```

#### OpenAI 对话示例

```
API 密钥：sk-xxxxxxxxxxxxxxxx
模型：gpt-4o
提示词：请解释一下人工智能的基本概念
温度：0.7
最大完成 tokens：1000
```

#### Sora 视频生成示例

```
API 密钥：sk-xxxxxxxxxxxxxxxx
模型：sora-2-pro
提示词：一个美丽的日落场景，海浪拍打着沙滩
时长（秒）：15
尺寸：1920x1080
水印：false
```

### 注意事项

1. 所有 API 调用均需有效的 API 密钥
2. 生成内容可能受版权和使用条款限制，请遵守相关规定
3. 视频生成可能需要较长时间，请耐心等待
4. 部分功能仅特定模型支持，请根据模型选择合适的参数
5. 参考图和角色视频链接需确保可访问，且不包含违规内容
